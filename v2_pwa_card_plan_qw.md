# План деплоя на сервер

## Этап 7: Деплой и Инфраструктура

### 7.1. Docker-контейнеризация
* **Цель:** Создание Dockerfile для развёртывания.
* **Файл:** `Dockerfile`
* **Логика:**
  - Base image: `python:3.13-slim`
  - Копирование requirements, установка зависимостей
  - Копирование кода приложения
  - CMD: `uvicorn app.main:app --host 0.0.0.0 --port 8000`

### 7.2. Docker Compose
* **Цель:** Оркестрация контейнеров (app + volumes для БД).
* **Файл:** `docker-compose.yml`
* **Логика:**
  - Сервис `app`: сборка из Dockerfile, порты, volumes для SQLite
  - **Примечание:** SQLite не требует отдельного контейнера, БД хранится в volume

### 7.3. Настройка Nginx + SSL
* **Цель:** HTTPS для PWA (обязательное требование).
* **Файл:** `nginx.conf`
* **Логика:**
  1. Reverse proxy на uvicorn (порт 8000)
  2. SSL-сертификат Let's Encrypt (Certbot)
  3. Редирект HTTP → HTTPS
  4. Настройка заголовков безопасности (HSTS, CSP)

### 7.4. Деплой на Selectel (иди отдельном сервере заказчика)
* **Цель:** Развёртывание на рабочем сервере.
* **Логика:**
  1. Клонирование репозитория на сервер
  2. Настройка `.env` с продакшен-параметрами
  3. Запуск через Docker Compose или systemd
  4. Настройка домена и SSL
