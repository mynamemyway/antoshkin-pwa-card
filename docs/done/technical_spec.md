# Техническое задание: Система лояльности "Digital Card" (PWA + SMS)

## 1. Общее описание

Создание прогрессивного веб-приложения (PWA) для автоматизации выдачи скидочных карт в 6 розничных точках. Система должна исключить сбор фейковых номеров и обеспечить простую установку на смартфоны клиентов без использования AppStore/Google Play.

## 2. Путь клиента (User Flow)

1. **Точка входа:** Клиент сканирует физический QR-код на кассе.
2. **Установка:** Открывается мобильная страница с приглашением "Установить карту лояльности".
3. **Автономность:** После добавления иконки, приложение открывается во весь экран (без интерфейса браузера).
4. **Идентификация:**
    * Ввод Имени и Номера телефона.
    * Нажатие кнопки "Получить карту".
5. **Верификация (SMS):**
    * На указанный номер приходит 4-значный код.
    * Клиент вводит код в приложении для подтверждения номера.
6. **Результат:** Отображение виртуальной карты с именем клиента, номером телефона и персональным QR-кодом для сканирования кассиром.

## 3. Технические требования

* **Backend:** Python 3.13 (FastAPI) — высокая скорость обработки запросов.
* **Frontend:** Адаптивный HTML5/CSS3 (Tailwind) — корректное отображение на всех типах смартфонов.
* **База данных:** SQLite/Postgres — хранение данных клиентов и статуса верификации.
* **СМС-шлюз:** Интеграция с сервисом (например, SMS.ru или аналог) по API.
* **Безопасность:** Работа через защищенный протокол HTTPS (обязательно для PWA).

## 4. Административная часть (Для заказчика)

* Хранение всех подтвержденных контактов в единой базе.
* Возможность выгрузки базы в формате Excel/CSV для дальнейшего маркетинга.

### Комментарии:

1. **Нюанс с "Кнопкой установить":** На iPhone (iOS) технически невозможно сделать одну кнопку "Установить". Мы должны показать клиенту небольшую визуальную подсказку: *"Нажмите 'Поделиться' -> 'На экран Домой'"*. На Android это работает автоматически через системный запрос.
2. **СМС-шлюз:** Нужно запросить у заказчика 500-1000 рублей на баланс СМС-сервиса для тестов и старта.
3. **QR-код бонусом:** Это отличный ход. Мы используем библиотеку `qrcode.js` прямо в браузере клиента — это не нагружает твой сервер и выглядит круто.
