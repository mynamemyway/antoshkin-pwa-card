# Баги и правки:

## Админ-панель

1. Требуется реализовать пагинацию (перелистывание страниц), когда верифицированных пользователей более 50ти - на проде их уже больше.
2. Поиск по номеру телефона работает только по показанным (последним) 50-ти пользователям - это некорректно. Администратор должен иметь возможность искать по всей базе номер телефона, а не только по последним 50.
3. Подсчёт верифицированных пользователей сейчас реализован по верифицированным только из 50-ти последних отображённых на странице, а должен показывать цифру всех верифицированных в базе.
