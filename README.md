# Antoshkin Loyalty Card (PWA)

Система цифровых дисконтных карт для цветочного комбината [rassada1.ru](https://rassada1.ru/).

## Ключевые функции
* **PWA (Progressive Web App):** Установка на смартфон в один клик без сторов.
* **SMS-верификация:** Защита от регистрации фейковых номеров.
* **QR-генерация:** Динамическое создание кода клиента на стороне браузера (qrcode.js).
* **Backend:** FastAPI (Python 3.13) + SQLite.
* **Frontend:** Tailwind CSS (адаптивный мобильный интерфейс).

## Стек (предварительный)
Python 3.13, FastAPI, SQLAlchemy, Jinja2, Tailwind CSS.
