## 📱 О проекте

PWA (Progressive Web App) для автоматизации выдачи скидочных карт в розничных точках. Система позволяет клиентам получать виртуальные карты лояльности через смартфон без установки приложения из AppStore/Google Play.

### Преимущества

- **Для клиентов:**
  - Установка в один клик через браузер
  - Не занимает место в памяти телефона
  - Всегда с собой в смартфоне
  - Мгновенная регистрация через SMS

- **Для бизнеса:**
  - Исключение фейковых номеров
  - Единая база клиентов
  - Экспорт данных для маркетинга
  - Работа без интернета (офлайн-режим)

---

## 🔑 Ключевые функции

* **PWA (Progressive Web App):** Установка на смартфон в один клик без сторов.
* **SMS-верификация:** Защита от регистрации фейковых номеров.
* **QR-генерация:** Динамическое создание кода клиента на стороне браузера (qrcode.js).
* **Сессии:** Автоматический вход при повторном открытии (30 дней).
* **Админ-панель:** Просмотр клиентов, экспорт в CSV, поиск по телефону.
* **Инструкция по установке:** Всплывающая подсказка для iOS и Android.

## 📊 Сценарии использования (User Flow)

```
Открытие приложения
    ↓
Есть ли активная сессия (cookie)?
    ├── ДА → Проверка в БД
    │         ├── Пользователь есть → Сразу показать карту
    │         └── Пользователя нет → Показать форму регистрации
    │
    └── НЕТ → Показать форму регистрации (имя + телефон)
              ↓
              Ввёл данные → Отправка SMS с кодом
              ↓
              Ввёл код → Верификация → Cookie (30 дней) → Карта
```

### Детальные сценарии

**Сценарий 1: Новый пользователь**
1. Открыл приложение → Форма регистрации
2. Ввёл имя и телефон → SMS с кодом
3. Ввёл код → Карта с QR-кодом

**Сценарий 2: Повторный вход (с сессией)**
1. Открыл приложение → Автоматический вход
2. Сразу видит карту

**Сценарий 3: Выход и повторный вход**
1. Нажал "Выйти" → Сессия удалена
2. Открыл приложение → Форма регистрации
3. Ввёл телефон → SMS с кодом
4. Ввёл код → Карта

**Сценарий 4: Установка PWA**
1. Открыл инструкцию (синий блок)
2. iOS: «Поделиться» → «Добавить на экран домой»
3. Android: «⋮» → «Добавить на гл. экран»
4. Иконка появилась на главном экране

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
# Создание виртуального окружения
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows

# Установка зависимостей
pip install -r requirements.txt
```

### 2. Настройка окружения

```bash
# Копирование шаблона
cp .env.example .env

# Редактирование .env
# SMS_API_KEY=your_api_id  # API ID от SMS.ru
# SMS_TEST_MODE=True       # True = тестовый режим (код "0000"), False = реальные SMS
```

### 3. Запуск приложения

```bash
# Запуск сервера разработки
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

Приложение доступно по адресу: `http://localhost:8000`

## 🧪 Тестирование

### Режим тестирования SMS

По умолчанию включён тестовый режим (`SMS_TEST_MODE=True`):
- **Код верификации:** `0000` (для любого номера)
- **SMS не отправляются** (код выводится в консоль)

### Боевой режим

Для отправки реальных SMS в `.env`:
```env
SMS_TEST_MODE=False  # Отправлять реальные SMS
SMS_API_KEY=your_api_id  # API ID от SMS.ru
```

**Требования:**
- Положительный баланс в SMS.ru
- Активированный API ключ в личном кабинете SMS.ru

### Чеклист для ручного тестирования

## 🔐 Безопасность

- **HttpOnly Cookie** — Токен сессии недоступен для JavaScript (защита от XSS)
- **Secure Cookie** — Передача только по HTTPS
- **SameSite=Lax** — Защита от CSRF-атак
- **Срок сессии** — 30 дней, затем требуется повторная авторизация
- **SMS-код** — 4 цифры, истекает через 5 минут
- **Токен сессии** — 256 бит энтропии (secrets.token_urlsafe)

## 📞 SMS-шлюз

### Интеграция с SMS.ru

**Настройка в `.env`:**
```env
SMS_API_KEY=your_api_id  # API ID из личного кабинета SMS.ru
SMS_TEST_MODE=True       # True = тестовый режим, False = реальные SMS
```

**Параметры SMS.ru API:**
- **Endpoint:** `https://sms.ru/sms/send`
- **Параметры:** `api_id`, `to`, `msg`, `json`
- **Текст SMS:** "Ваш код верификации: 1234"

**Требования:**
- ✅ Положительный баланс в SMS.ru
- ✅ Активированный API ключ в личном кабинете
- ✅ Номер телефона в формате `+7XXXXXXXXXX`
